# Personna-MVVM-Map-Coroutines-LiveData

 1- Show Random list of users using restful [API](https://randomuser.me/api/?results=10) 
 2- Show Random markers of location users on Mapbox using Coroutines. 


### Features
- Kotlin language
- MVVM Architecture
- Retrofit
- Coroutines
- LiveData
- Mapbox
- Picasso
- LifeCycle
