# Personna-MVVM-Map-Coroutines-LiveData

- Show Random list of users using restful [API](https://randomuser.me/api/?results=10) 
- Show Random markers of location users on Mapbox using Coroutines. 


### Features
- Kotlin language
- MVVM Architecture
- Retrofit
- Coroutines
- LiveData
- Mapbox
- Picasso
- LifeCycle
